PRIVACY POLICY

Effective as of Januaury 29,2022



Data Storage

We store datas such as server ID, user ID, role ID and channel ID only for the purpose of the functionality of some features of the bot such as server logs, auto-role and backup purposes. None of the data is publicly avalable and no other data other than these are stored.


Use Of Collected Information

The collected datas are used for backup purposes and for the functionality of some features such as server logs, auto-role. These collected datas are stored only for a limited time.


Error Data Storage

Error logs of the bot are stored only or the purpose of fixation purposes. Note that these error datas are only accessible by the developers of the bot.


Security Of The Stored Data

The security of the users data are important for us and hence the datas are stored safely in a password secured database and are accessible only by the developers of the bot.


Data Sharing

None of the user and server data would be shared with any third-party services for any reason and these datas are not publicly avalable and are stored in a password secured database.


Deletion of Data

Users can request deletion of their data by sending us an email to cactusbot21@gmail.com or by contacting one of the developer of the bot through the support server. Thereafter the deletion process will be carried out.

Changes To This Policy

At our discretion, we may change our privacy policy to reflect updates to our service processes, current acceptable practices, or legislative or regulatory changes. If we decide to change this privacy policy, we will post the changes here at the same link by which you are accessing this privacy policy. If required by law, we will get your permission or give you the opportunity to opt in to or opt out of, as applicable, any new uses of your personal information
