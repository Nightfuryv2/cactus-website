Features Of Cactus

High Performance

The bot has 24/7 uptime with an average of 99% uptime and it never lags!

Logging System

The bot has an advanced logging system logging every changes in your server including role logs, join-leave logs, message logs, thread logs, etc.

Custom Tags System

The bot has a custom tag system which helps you to make custom tags for your server!

Information

The bot has some info commands that can give information on roles, servers, users, covid stats, etc.

Utility

Being a multipurpose bot, the bot helps you in moderating your server and has tons of fun and utility commands!

Games

Play games such as quick click, snake and minesweeper! More games to be released in the future updates!
