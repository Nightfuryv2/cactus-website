Updates

Cactus version 3.2.5 is getting updated to version 4.0.0 soon! In this update, prefixed commands are returning back again to the bot! After this update the bot will have both prefixed as well as slash commands! The bot will have updates released slowly one by one! So here are the list of things that you can expect in the version 4:

  • Auto-moderation system
  • Prefixed commands
  • Command disable and enable system
  • Neater logging system
  • Game commands
  • More info and utility commands
  • Image generation commands
  • Voice channel activites to enjoy!
  • Notes system

Expect this update to be a stable one which will be maintained until the bot gets verified! After the growth being dead for 2-3 months, we have finally seen a great growth in the past 2 months, we gained 12 servers in the past 15 days! Thanks to Dank space for being the first big server to use our bot often! Thank you so much to everyone who supported us in our journey! This update of v4 will be the biggest update ever done and therefore will take a lots of time to finish it, almost all of the commands have been made into prefixed commands, we just need to change few more commands to prefixed commands and add the new features to it! Expected time of release of the full stable update is in sometime in the middle of next year! Once again thanks for all the support!
